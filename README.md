
# Bhai Bhai Group of Companies

> A premium business portfolio website showcasing 56+ businesses across 10 categories.

🌐 **Live Demo:** [awaissaeed-eng.github.io/Bhai-bhai-group-of-companies](https://awaissaeed-eng.github.io/Bhai-bhai-group-of-companies/)

## Tech Stack
- HTML5
- Tailwind CSS
- Vanilla JavaScript
- EmailJS (contact form)
- Google Maps Embed

## Features
- 10 business categories
- Individual business detail pages
- Google Maps integration
- WhatsApp direct link
- Contact form with email
- Fully responsive design
